# 💱 Currency Converter (React)

A fast and responsive currency converter built using **React**, **Vite**, and **Tailwind CSS** that fetches real-time exchange rates.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/KhushiB03/Currency-Converter.git
```

2. Navigate to the project folder:

```bash
cd Currency-Converter-React
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

---

## ⚙️ How It Works

* Fetches real-time exchange rates from Frankfurter API
* Uses a custom React hook (`useCurrencyInfo`)
* Updates conversion instantly when user changes currency

---

## 🌍 API Used

* https://api.frankfurter.app
* Provides real-time currency exchange rates

---

## 💡 Example

100 USD → INR ≈ ₹8300 *(varies based on live rates)*
